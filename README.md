# 🤖 Hetzner Manager Bot & Web Panel

یک ربات پیشرفته تلگرامی به همراه پنل مدیریت تحت وب برای مدیریت و فروش خودکار سرورهای ابری هتزنر (Hetzner Cloud). 

این پروژه به شما اجازه می‌دهد تا صفر تا صد مدیریت سرورها، سیستم فروش، و درگاه‌های پرداخت را از طریق یکپارچه‌سازی ربات تلگرام و یک پنل وب اختصاصی مدیریت کنید.

## ✨ امکانات

- 🖥 **مدیریت کامل سرورها:** امکان ساخت، حذف، روشن/خاموش کردن، ری‌استارت و ریست پسورد سرور مستقیماً از تلگرام.
- 🌐 **تغییر آی‌پی (Change IP):** قابلیت درخواست تغییر آی‌پی توسط کاربران و کسر خودکار هزینه از کیف پول آن‌ها.
- 💰 **کیف پول و سیستم مالی:** پشتیبانی از شارژ حساب کاربری با ارزهای دیجیتال (Tether TRC20 و غیره) و درگاه‌های پرداخت ایرانی.
- 🎛 **پنل مدیریت تحت وب:** پنل وب با رابط کاربری اختصاصی (Sky Blue و باکس‌های شیشه‌ای) برای تنظیم قیمت‌ها، موجودی‌ها و لوکیشن‌های فعال.
- 🔗 **دسترسی آسان:** ورود به پنل مدیریت مستقیماً از طریق دکمه‌های شیشه‌ای (Inline Buttons) در ربات تلگرام.
- ⚙️ **نصب خودکار:** راه‌اندازی سریع تمام پیش‌نیازها و سرویس‌ها تنها با یک دستور.

## 🚀 راه‌اندازی و نصب سریع

برای نصب یا بروزرسانی ربات و پنل روی سرور لینوکسی (پیشنهاد: اوبونتو 20.04 یا 22.04)، فقط کافیست دستور زیر را کپی کرده و در ترمینال سرور خود اجرا کنید:

bash <(curl -s [https://raw.githubusercontent.com/Aziz-dev22/hetzner_bot/main/install.sh](https://raw.githubusercontent.com/Aziz-dev22/hetzner_bot/main/install.sh))
